[Whisper](https://openai.com/research/whisper) is an automatic speech recognition system that enables transcription in multiple languages.

We provide the [**mosec**](../mosec) inference service template.

## Mosec

You can deploy the pre-built template [here](https://cloud.modelz.ai/deployment/template) to deploy the service. Run the inference with [**SDK**](../../sdk).

- CLI
```shell copy
export MODELZ_API_KEY=mzi-abcdefg...
modelz inference --deployment whisper-demo-XXX --serde raw --read-stdin <speech_16000.mp3
```

- Python Client
```python copy
import modelz


APIKey = "mzi-abcdefg..."

cli = modelz.ModelzClient(deployment="whisper-demo-XXX", key=APIKey)
with open("speech_16000.mp3", "rb") as f:
    data = f.read()
resp = cli.inference(params=data, serde="raw")
print(resp.data)
```
