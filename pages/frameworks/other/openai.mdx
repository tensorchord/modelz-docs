# OpenAI Compatible API for Open Source LLMs

We provide an OpenAI compatible API for open source LLMs with the help of [Modelz-LLM](https://github.com/tensorchord/modelz-llm). You could easily integrate new open source LLMs (e.g. [LLaMA](https://github.com/facebookresearch/llama), [ChatGLM](https://github.com/THUDM/ChatGLM-6B/blob/main/README_en.md)...) into your existing applications without any code changes if you are using [OpenAI python package](https://github.com/openai/openai-python) or [langchain](https://github.com/hwchase17/langchain).

## Features

- **OpenAI compatible API**: Modelz LLM provides an OpenAI compatible API for LLMs, which means you can use the OpenAI python SDK to interact with the model.
- **Self-hosted**: Modelz LLM can be easily deployed on either local or cloud-based environments.
- **Open source LLMs**: Modelz LLM supports open source LLMs, such as FastChat, LLaMA, and ChatGLM.
- **Cloud native**: We provide docker images for different LLMs, which can be easily deployed on Kubernetes, or other cloud-based environments (e.g. [Modelz](https://docs.modelz.ai))

## Use OpenAI python SDK

Then you can use the OpenAI python SDK to interact with the model:

```python
import openai
openai.api_base="https://<deployment-key>.modelz.io"
openai.api_key="mzi-xxx"

# create a chat completion
chat_completion = openai.ChatCompletion.create(model="any", messages=[{"role": "user", "content": "Hello world"}])
```

## Supported APIs

- `/completions`
- `/chat/completions`
- `/embeddings`
- `/engines/<any>/embeddings`
- `/v1/completions`
- `/v1/chat/completions`
- `/v1/embeddings`
