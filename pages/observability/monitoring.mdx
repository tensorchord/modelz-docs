# Monitoring

We provide a dashboard that allows you to visualize, explore, and monitor your usage and traffic. You can use the dashboard to monitor the following:

- **Total replicas**: The number of total replicas.
- **Available replicas**: The number of replicas that are currently available to serve requests.
- **Inflight requests**: The number of requests that are currently being processed.
- **Total requests**: The total number of requests that have been processed.
